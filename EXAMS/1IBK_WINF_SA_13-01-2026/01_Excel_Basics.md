## Cell References

### English
> **DEFINITION:** A cell reference indicates the location of a cell or a range of cells in a worksheet.

### Deutsch
> **DEFINITION:** Ein Zellbezug gibt den Speicherort einer Zelle oder eines Zellbereichs in einem Arbeitsblatt an.

### Types of References

| Type | Example | Behavior When Copied |
|------|---------|---------------------|
| **Relative** | `A1` | Changes (A1 ‚Üí B1 ‚Üí C1) |
| **Absolute** | `$A$1` | Stays constant |
| **Mixed (Column Fixed)** | `$A1` | Column stays, row changes |
| **Mixed (Row Fixed)** | `A$1` | Row stays, column changes |

### How References Change When Copied

**Relative Reference `A1`:**
- Copy right: A1 ‚Üí B1 ‚Üí C1
- Copy down: A1 ‚Üí A2 ‚Üí A3

**Absolute Reference `$A$1`:**
- Copy anywhere: Always stays $A$1

**Mixed Reference `$A1`:**
- Copy right: $A1 ‚Üí $A1 ‚Üí $A1 (column locked)
- Copy down: $A1 ‚Üí $A2 ‚Üí $A3 (row changes)
