1. Einführung / Introduction

English

The passive voice in German is a grammatical construction used to emphasize the action or the result of an action, rather than the agent. It is often referred to as the "Leideform" (suffering form) because the subject of the passive sentence undergoes the action.

This mode is particularly useful when the agent is unknown, unimportant, or intentionally not mentioned—for example, in news reports or formal texts.

Deutsch

Das Passiv im Deutschen ist eine grammatische Konstruktion, die verwendet wird, um die Handlung oder das Ergebnis einer Handlung hervorzuheben, und nicht den Handelnden. Es wird oft als „Leideform" bezeichnet, weil das Subjekt des Passivsatzes die Handlung erleidet.

Diese Form ist besonders nützlich, wenn der Handelnde unbekannt, unwichtig oder absichtlich nicht genannt wird, zum Beispiel in Nachrichtenberichten oder formellen Texten.

2. Basic Passive Voice (B1 Level)

Formation of Vorgangspassiv / Bildung des Vorgangspassivs

Vorgangspassiv (Process Passive)

EN: Describes an ongoing process or action. Focuses on what is being done. Formation: werden + Partizip II

DE: Beschreibt einen laufenden Vorgang. Konzentriert sich auf das, was getan wird. Bildung: werden + Partizip II

Vorgangspassiv in Different Tenses / in verschiedenen Zeitformen

Zeitform / Tense Aktiv (Active) Passiv (Passive)
Präsens Der Mechaniker wechselt den Reifen. Der Reifen wird gewechselt.
Präteritum Der Mechaniker wechselte den Reifen. Der Reifen wurde gewechselt.
Perfekt Der Mechaniker hat den Reifen gewechselt. Der Reifen ist gewechselt worden.
Plusquamperfekt Der Mechaniker hatte den Reifen gewechselt. Der Reifen war gewechselt worden.
Futur I Der Mechaniker wird den Reifen wechseln. Der Reifen wird gewechselt werden.
Futur II Der Mechaniker wird den Reifen gewechselt haben. Der Reifen wird gewechselt worden sein.

When to Use Passive Voice / Wann man das Passiv verwendet

English
  • The agent (doer) is unknown or unimportant
  • Focus is on the action or result, not who performed it
  • In formal writing, academic texts, instructions, or procedures
  • The agent is obvious from context
Deutsch
  • Der Handelnde ist unbekannt oder unwichtig
  • Fokus liegt auf der Handlung oder dem Ergebnis
  • In formeller Schriftsprache, akademischen Texten, Anweisungen
  • Der Handelnde ist aus dem Kontext offensichtlich

Examples / Beispiele

  • Das Auto wird repariert. — The car is being repaired.
  • Die Prüfung wurde bestanden. — The exam was passed.
  • Das Problem ist gelöst worden. — The problem has been solved.

3. Active to Passive Transformation

English
  1. The direct object of the active sentence becomes the subject of the passive sentence.
  2. The auxiliary verb werden is conjugated according to the tense and the new subject.
  3. The main verb is converted to its Partizip II form.
  4. The original subject (agent) can be omitted or introduced with von (people) or durch (things/means).
Deutsch
  1. Das Akkusativobjekt des Aktivsatzes wird zum Subjekt des Passivsatzes.
  2. Das Hilfsverb werden wird entsprechend der Zeitform und dem neuen Subjekt konjugiert.
  3. Das Hauptverb wird in seine Partizip II-Form gebracht.
  4. Das ursprüngliche Subjekt kann weggelassen oder mit von (Personen) oder durch (Sachen) eingeführt werden.
Transformation Examples / Transformationsbeispiele

Aktiv: Der Mechaniker repariert das Auto.
Passiv: Das Auto wird von dem Mechaniker repariert.

Aktiv: Die Lehrerin erklärt die Grammatik.
Passiv: Die Grammatik wird von der Lehrerin erklärt.

Examples by Context / Anwendungsbeispiele nach Kontext

Daily Life / Alltag

  • Das Frühstück wird um 8 Uhr serviert.
  • Die Post wird jeden Morgen gebracht.
  • Das Zimmer wird täglich gereinigt.

Work / Arbeit

  • Die Berichte werden wöchentlich geschrieben.
  • Die E-Mails werden sofort beantwortet.
  • Die Meetings werden montags geplant.

School / Schule

  • Die Prüfungen werden nächste Woche geschrieben.
  • Die Hausaufgaben werden jeden Tag korrigiert.
  • Die Bücher werden von der Bibliothek ausgeliehen.

4. Advanced Passive Voice (B2/C1 Level)

Vorgangspassiv vs. Zustandspassiv

Vorgangspassiv (Process)

Describes an action in progress.
Formation: werden + Partizip II
Das Auto wird repariert. (is being repaired)

Zustandspassiv (State)

Describes a resulting state.
Formation: sein + Partizip II
Das Auto ist repariert. (is repaired)

Zustandspassiv in Different Tenses

Zeitform Example Explanation
Präsens Der Reifen ist gewechselt. The tire is (already) changed.
Präteritum Der Reifen war gewechselt. The tire was (already) changed.
Futur I Der Reifen wird gewechselt sein. The tire will be (already) changed.

Examples / Beispiele

  • Das Buch ist geschrieben. — The book is written (state).
  • Die Tür ist geöffnet. — The door is open (state).
  • Das Essen ist gekocht. — The food is cooked (state).

Agent in Passive: "von" and "durch"

von + Dativ / durch + Akkusativ

von: For people or institutions.
Das Buch wurde von dem Autor geschrieben.

durch: For things, means, or causes.
Das Problem wurde durch neue Technologie gelöst.

Impersonal Passive ("Es" Passive)

English

Used when there is no specific subject, particularly with intransitive verbs.

  • Es wird getanzt. — There is dancing.
  • Gestern wurde viel gearbeitet. — Much work was done yesterday.
Deutsch

Wird verwendet, wenn kein spezifisches Subjekt vorhanden ist, insbesondere bei intransitiven Verben.

  • Es wird getanzt.
  • Gestern wurde viel gearbeitet. („Es" kann weggelassen werden)

Passive with Modal Verbs

Formation / Bildung

Modal verb + Partizip II + werden (infinitive)

  • Das muss gemacht werden. — That must be done.
  • Das kann nicht geändert werden. — That cannot be changed.
  • Die Hausaufgaben sollen gemacht werden. — The homework should be done.
  • Die Tür darf geöffnet werden. — The door may be opened.

Passive in Subordinate Clauses

Complex Constructions / Komplexe Konstruktionen
  • Ich weiß, dass das Auto repariert wird.
  • Es ist wichtig, dass die Regeln befolgt werden.
  • Die Frage ist, ob das Problem gelöst werden kann.

Circumscribing Passive with „man"

Instead of passive, use man or jemand as subject:

  • Passive: Der Reifen wird gewechselt.
  • Alternative: Man wechselt den Reifen.

5. Common Mistakes / Häufige Fehler

1. Confusing Vorgangspassiv and Zustandspassiv

✗ Incorrect: Das Auto ist repariert worden. (for state)

✓ Correct: Das Auto ist repariert. (state passive)

"ist repariert worden" = Perfekt Vorgangspassiv; "ist repariert" = Zustandspassiv

2. Wrong Word Order / Falsche Wortstellung

✗ Incorrect: Wird das Auto repariert von dem Mechaniker.

✓ Correct: Das Auto wird von dem Mechaniker repariert.

3. Missing "von" or "durch"

✗ Incorrect: Das Buch wird dem Autor geschrieben.

✓ Correct: Das Buch wird von dem Autor geschrieben.

4. Wrong Tense Formation (Perfekt)

✗ Incorrect: Das Auto ist repariert werden.

✓ Correct: Das Auto ist repariert worden.

5. Wrong Modal Verb Placement

✗ Incorrect: Das Auto repariert werden muss.

✓ Correct: Das Auto muss repariert werden.

6. Missing "werden" with Modal Verbs

✗ Incorrect: Das Auto muss repariert.

✓ Correct: Das Auto muss repariert werden.

Exercise 5: Modal Passive (C1)

  1. Man muss die Regeln befolgen. → Die Regeln müssen befolgt werden.

  1. Ich weiß, dass das Auto ___ (reparieren). → repariert wird
  2. Es ist wichtig, dass die Regeln ___ (befolgen). → befolgt werden
  3. Die Frage ist, ob das Problem ___ (lösen). → gelöst werden kann

7. Memory Tips / Merk-Tipps

1. Learn the Formation Patterns

werden + Partizip II = Vorgangspassiv
sein + Partizip II = Zustandspassiv

2. Practice Transformations

Convert active to passive and vice versa in all tenses regularly.

3. Use Appropriate Agents

von for people → durch for things/means

4. Key Question: Action or State?

Action happening? → werden (Vorgangspassiv)
State achieved? → sein (Zustandspassiv)

5. Active → Passive Transformation Example

Example / Beispiel

Aktiv: Der Lehrer korrigiert die Hausaufgaben.
Passiv: Die Hausaufgaben werden (vom Lehrer) korrigiert.

Steps: 1) Akkusativobjekt → Subjekt, 2) werden konjugieren, 3) Partizip II am Ende, 4) optional: von + Dativ

Practice: Transform Active to Passive

Transform these active sentences to passive voice:

  1. Die Schüler lesen das Buch. → Das Buch wird (von den Schülern) gelesen.
  2. Der Koch bereitet das Essen zu. → Das Essen wird (vom Koch) zubereitet.
  3. Die Firma baut ein neues Gebäude. → Ein neues Gebäude wird (von der Firma) gebaut.
  4. Der Arzt untersucht den Patienten. → Der Patient wird (vom Arzt) untersucht.
  5. Die Sekretärin beantwortet die E-Mails. → Die E-Mails werden (von der Sekretärin) beantwortet.
